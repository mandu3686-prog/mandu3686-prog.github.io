## 🏗️ 이 사이트를 만든 방법: 첫 번째 프로젝트

이 포트폴리오 사이트는 정적 사이트 생성기(Static Site Generator)를 활용한 웹사이트 구축 과정을 학습하기 위한 첫 번째 프로젝트입니다.

### 📌 사용 기술 및 환경
* **배포 플랫폼**: GitHub Pages (정적 웹사이트 호스팅)
* **프레임워크**: Jekyll (Markdown 기반의 정적 사이트 생성기)
* **테마**: So Simple Theme (YAML/Markdown 문법 학습을 위한 미니멀 테마 선택)
* **주요 사용 파일**: `_config.yml` (사이트 설정), `index.md` (메인 콘텐츠), `assets/css/style.css` (색상 커스터마이징)

### 💡 제작 과정에서 배운 핵심
| 학습 주제 | 구체적 경험 및 배운 점 |
| :--- | :--- |
| **YAML 문법** | `_config.yml` 설정 시 콜론 뒤 공백 누락 등으로 빌드 실패(404 에러)를 반복 경험했습니다. 이 과정을 통해 **문법의 정확성**과 **디버깅 능력**의 중요성을 체감했습니다. |
| **테마 활용** | `remote_theme` 설정만으로 사이트 디자인을 변경하고, CSS 파일을 추가하여 **기존 테마 위에 새로운 스타일을 덮어쓰는** 방법을 학습했습니다. |
| **버전 관리** | 사이트 제작 초기부터 Git의 커밋(Commit)과 푸시(Push)를 통해 **변경 이력 관리**를 습관화했습니다. |

## 💼 주요 학습 결과물


1. 📋순수 JavaScript 기반 TODO List
이 프로젝트는 Vanilla JavaScript를 사용하여 DOM 조작과 데이터 영속성 처리 능력을 입증하는 데 중점을 둡니다.

학습 목표: 순수 JavaScript를 이용한 DOM 조작 숙련 및 데이터 영속성 확보

사용 기술: HTML5, CSS3, Vanilla JavaScript (ES6+), localStorage

💡 학습 과정 및 성과
핵심 기능 구현: 사용자의 할 일 목록(Todo Item)을 추가, 삭제, 완료 상태 토글하는 기능을 getElementById, createElement 등의 메서드를 사용하여 직접 구현했습니다.

데이터 관리: 브라우저의 localStorage API를 사용하여 사용자가 페이지를 닫았다가 다시 열어도 목록 데이터가 유지되도록 구현했습니다. 이 과정에서 **JSON.stringify()**와 **JSON.parse()**를 이용한 데이터 직렬화/역직렬화 방법을 학습했습니다.

🧗‍♀️ 난관과 해결 (성장 기록)
문제: 목록의 특정 항목을 삭제할 때, 어떤 버튼이 눌렸는지 식별하고 해당 항목만 삭제하는 로직 구현이 어려웠습니다.

해결: event.target 속성을 이용하여 클릭된 요소를 정확히 파악하고, closest() 메서드를 사용하여 해당 버튼의 가장 가까운 부모 요소(목록 아이템)를 찾아 제거하는 방법으로 해결했습니다. 이를 통해 **이벤트 위임(Event Delegation)**의 개념을 이해하게 되었습니다.

---

2. 🎨모바일 우선 반응형 웹페이지 구현
이 프로젝트는 HTML/CSS의 기본기를 점검하고, 모든 디바이스에서 깨지지 않는 웹사이트를 만드는 능력을 보여줍니다.

학습 목표: 웹 표준을 준수하는 시맨틱 마크업과 모바일 우선 반응형 웹 설계

사용 기술: HTML5, CSS3 (Flexbox, Grid), Media Query

💡 학습 과정 및 성과
레이아웃 구현: 복잡한 헤더와 푸터를 Flexbox를 사용하여 정렬하고, 메인 콘텐츠 영역을 CSS Grid를 활용하여 구성하며 두 레이아웃 모델의 장단점을 파악했습니다.

반응형 설계: @media 쿼리를 활용하여 화면 너비(768px, 480px)를 기준으로 CSS 스타일을 다르게 적용했습니다. **max-width 대신 min-width**를 사용하는 모바일 우선(Mobile First) 디자인 원칙을 실습했습니다.

🧗‍♀️ 난관과 해결 (성장 기록)
문제: 데스크톱에서 정상적으로 보이던 이미지가 모바일에서 화면을 벗어나 스크롤을 유발했습니다.

해결: 이미지 태그에 width: 100%;와 height: auto;를 적용하여 가로 길이에 맞춰 높이가 자동으로 조절되도록 설정했습니다. 이로써 이미지 크기 처리의 기본적인 반응형 원칙을 이해하게 되었습니다.
